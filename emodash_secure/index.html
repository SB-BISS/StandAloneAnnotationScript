Welkom!


U gaat luisteren naar call center gesprekken. U krijgt telkens fragmenten van 3 seconden uit deze gesprekken te horen.

Wij vragen u goed te luisteren naar elk fragment en daarna de emotie die u in het fragment gehoord heeft aan te geven. U kunt meerdere emoties tegelijkertijd aangeven per fragment, ook kunt u “neutraal” selecteren als er geen specifieke emotie te horen was in het fragment.

Hieronder ziet u een lijst van emoties waar u uit kunt kiezen.

Boosheid
Voorbeeld: Klant is ontevreden met het antwoord -> klant reageert boos
Angst
Voorbeeld: Klant krijgt tegenvallende pensioen informatie -> klant is bang
Afkeer
Voorbeeld: Klant reageert minachtend naar de agent -> klant voelt afkeer tegenover de agent
Verdriet
Voorbeeld: Klant maakt een beledigend opmerking -> agent is bedroefd
Blijdschap
Voorbeeld: Agent lost een problem op -> klant is blij
Verbazing/verrassing
Voorbeeld: Agent geeft onverwacht nieuws -> klant is verbaasd
Neutraal:
Er is geen specifieke emotie te horen in het fragment

Klik op START als u klaar bent om naar het eerste fragment te luisteren.
